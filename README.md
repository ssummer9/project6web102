# Web Development Project 6 - *Studio Ghbili Dashboard*

Submitted by: **Sawaira Summer**

This web app: **In this website introduces Studio Ghibli movies made throughout the years with a line graph and bar graph with also a description of each movie when clicked on the movie's title (also represented as the URL). **

Time spent: **8** hours spent in total
